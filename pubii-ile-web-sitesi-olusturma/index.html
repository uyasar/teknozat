Publii İle Süper Hızlı Web Sitesi Nasıl Açılır?

U. Yazar Ocak 25, 2025

Publii bir statik web sitesi oluşturucu diyebileceğimiz bir içerik yönetim sistemi. Diğer içerik yönetim sistemlerinin aksine, dosyaları bir sunucuya yükleyip cms'yi oradan yönetmiyoruz. Bilgisyaraımıza kurduğumuz bir program vasıtasıyla sitemizi hazırlayıp bunu istediğimiz yerde barındırıyoruz. Bir çok ücretsiz seçenek mevcut. 

Bu sistemin olumlu tarafı html tabanlı sayfalar ürettiği için çok hızlı ve seo uyumlu. Ayrıca herhangi bir veritabanı ve ya dinamik dosya barındırmadığı için daha güvenli.

Olumlu Tarafları

  • Çok hızlı
  • Seo uyumlu
  • Ücretsiz
  • Kullanışlı

Olumsuz Tarafları

  • Dinamik içerik oluşturulamaz
  • Tema sayısı sınırlı

Publii İle site oluşturma

  1. Getpublii.com üzerinden uygulamamızı indiriyoruz. Uygulama mac, windows ve linux olmak üzere tüm masaüstü işletim sistemlerinde çalışıyor.
  2. Uygulamayı açıp ilk sitemizi oluşturuyoruz.
  3. Github üzerinden hesap açıp bir repo oluşturuyoruz. Repomuzu public yapıp, pages üzerinden özel alanadı bağlıyoruz.
  4. Publii uygulamamıza gelip github verilerimizi server ayarları sekmesinden giriyoruz. 
  5. Yazılarımızı yazıp, yayına alıyoruz. 

Oldukça kolay.

U. Yazar
